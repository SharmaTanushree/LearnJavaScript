## Objects

- create object using object literal;
  ```js
  let user = {
    id : ~~(Math.random()*100),
    name : 'Michael'
  }
  ```
- create property name 'last name'
  
  ```js
  let user = {
   id : ~~(Math.random()*100),
  'First name' : 'Michael'
  };
  user['First name'];          //Michael
  ```
